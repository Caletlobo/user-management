Post-Lab Questions & Answers
QUESTION 1
How to check index usage using .explain() in Postman?
Add this route to server.js:

app.get('/users/explain', async (req, res) => {
  const result = await User.find({ name: req.query.name }).explain('executionStats');
  res.json({ keysExamined: result.executionStats.totalKeysExamined, ... });
});
Then in Postman: GET http://localhost:3000/users/explain?name=Alice
QUESTION 2
Compound index { email: 1, age: -1 } — which queries use it?
✓
find({ email: "test@gmail.com" })
USES index — email is the leftmost prefix field.
✗
find({ age: 25 })
DOES NOT use index — age alone skips the prefix. MongoDB uses COLLSCAN.
✓
find({ email: "test@gmail.com", age: 25 })
USES index — both fields present, full compound index utilized.
QUESTION 3
Missing vs Duplicate email — same error?
POST without email
Mongoose Validation Error — caught BEFORE the database. Error: "Path 'email' is required"
POST with duplicate email
MongoDB E11000 Error — caught AT the database. Error: "duplicate key error collection"
They are DIFFERENT errors at different layers: Mongoose validation (before DB) vs MongoDB constraint (at DB). Different error codes, different messages.
